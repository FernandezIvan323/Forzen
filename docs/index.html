Tres pasos simples para empezar

1

DESCARGA E INSTALA

Descarga el instalador .msi y ejecútalo. No necesitas Java — el runtime viene incluido.

2

ABRE FORZEN

Aparecerá un icono verde en la bandeja del sistema. La lupa ya está activa.

3

CONTROLA CON TECLAS

Usa Ctrl+Alt+↑ para aumentar el zoom. Cambia de modo con Ctrl+Alt+M.
