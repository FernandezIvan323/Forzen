## Funcionalidades

| | |
|---|---|
| 🔍 | Zoom 1x–8x con lupa que sigue el cursor |
| ⚡ | Captura a 60 FPS con aceleración GPU |
| 🎨 | Filtros: invertir, alto contraste, grises, daltonismo |
| ⌨️ | Atajos globales de teclado |
| 🖥️ | Modos: Lens, Full-Screen, Docked |
| 📝 | OCR + aumento de texto selectivo *(próximamente)* |
| 🎤 | Control por voz *(próximamente)* |

---

## Atajos de teclado

| Acción | Teclas |
|---|---|
| Zoom + | `Ctrl + Alt + ↑` |
| Zoom - | `Ctrl + Alt + ↓` |
| Pausar / Reanudar | `Ctrl + Alt + Z` |
| Cambiar modo | `Ctrl + Alt + M` |
| Abrir ajustes | `Ctrl + Alt + ,` |
| Salir | `Ctrl + Alt + X` |

---

## Compilar desde fuente

```powershell
# Requisitos: JDK 21 + Maven 3.9+
cd Forzen
mvn package
mvn javafx:run    # para ejecutar
```
